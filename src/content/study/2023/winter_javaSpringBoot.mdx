# 스터디 목표
- Java & Spring Boot 를 활용하여 간단한 게시판을 구현 해 봅니다.
- 스터디에 필요한 사전 지식
	- Java에 대한 기초 지식
	- HTML, CSS, JavaScript 기초 지식
	- HTTP에 대한 기초 지식
	- RDBMS에 대한 기초 지식
- 본스터디는간단한게시판구현,스프링부트프로젝트의전체적인흐름과틀을실습을통해알아 보기 위함이 목적입니다.
- 스터디 중 사용하는 외부 라이브러리 , Spring Security, 툴 관련된 내용들은 별도 키워드를 알려드려 학습 방향을 제안 해 드릴게요, 물론 학습 하시면서 질문은 언제든지 해 주시면 됩니다!


# 스터디 방식

- 구현하게될게시판은“점프투스프링부트”서적을참고,필요한부분만발췌하여개발환경설정부터 AWS 배포까지 스프링 부트 프로젝트를 실습해보는 스터디를 진행합니다.
- 질문이 있을때는 자유롭게 해주세요!
	- 혼자서 구현하고 있는 웹 어플리케이션 프로젝트 또는 백엔드 스택 관련 질문들도 언제든지 괜찮으니 자유롭게 해주세요!
- 추가 자세한 사항은 오리엔테이션 및 스터디 진행 하며 설명 드릴게요.
